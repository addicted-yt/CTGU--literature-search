--------------------------------
一、项目简介
--------------------------------
本工具用于辅助科研文献调研，自动完成以下流程：

1. 从控制台页面输入学号、密码、检索关键词、抓取条数和检索模式
2. 跳转三峡大学图书馆 → 统一身份认证登录
3. 根据模式自动访问：
   - 中国知网（CNKI）
   - Elsevier ScienceDirect（爱思唯尔）
4. 自动执行检索、采集列表信息和摘要
5. 一键生成带格式的 Word 文献调研报告（.doc）

所有账号密码只在本地浏览器中使用，不会发送到第三方服务器。

--------------------------------
二、文件说明
--------------------------------
1. CTGU_Console.html
   - 启动控制台页面（用户界面）
   - 用于输入账号、密码、关键词、抓取条数和选择模式
   - 点击按钮后会在新标签页中打开图书馆页面，触发脚本工作

2. 三峡大学科研全自动收割机-V12.4.js
   - Tampermonkey（油猴）用户脚本
   - 在以下页面自动运行：
     * ids.ctgu.edu.cn（统一身份认证登录页）
     * lib.ctgu.edu.cn（三峡大学图书馆）
     * *.cnki.net（知网）
     * *.sciencedirect.com 或 61.136.151.252:7070（学校内网的 SD 代理）
   - 完成登录、检索、采集、生成 Word 的全过程

--------------------------------
三、环境要求
--------------------------------
- 浏览器：推荐 Chrome / Edge
- 插件：Tampermonkey（油猴）扩展（可能也叫篡改猴）
- 网络：能够正常访问学校图书馆、统一身份认证、知网和 ScienceDirect

--------------------------------
四、安装步骤
--------------------------------
1. 安装 Tampermonkey 扩展
   - 打开浏览器扩展商店
   - 搜索 “Tampermonkey”或“篡改猴”
   - 安装后在浏览器右上角确认图标已显示

2. 导入脚本
   - 点击“Tampermonkey”或“篡改猴”图标 → 选择“管理面板” → 右上角选择“实用工具”
   - 在导入处“选择文件”中 选择 .js文件 打开
   - 选择安装

3. 准备控制台页面
   - 本地打开
     * 将 CTGU_Console.html 放在任意本地目录
     * 双击或拖入浏览器打开

--------------------------------
五、控制台使用说明（CTGU_Console.html）
--------------------------------
页面打开后，依次填写：

1. 统一身份认证（学号）
   - 输入本人学号（如：2020XXXXXX）

2. 认证密码
   - 输入统一认证密码
   - 右侧“显示/隐藏”按钮可检查输入

3. 关键词（中/英）
   - 中文搜索词：用于知网检索
   - English Keywords：用于 ScienceDirect 检索
   - 下方有若干示例标签可一键填入

4. 抓取论文条数（1~20）
   - 输入 1~20 的整数，表示每个站点最多抓取多少篇文章
   - 不填则默认 10

5. 检索模式（三个按钮）
   - 🔄 知网 + 爱思唯尔：先 CNKI 后 ScienceDirect，生成合并报告
   - 📚 仅知网：只采集 CNKI，并生成仅包含知网部分的报告
   - 🌍 仅爱思唯尔：只采集 ScienceDirect，并生成仅包含 SD 部分的报告

点击任一按钮后，将自动打开新的图书馆标签页，后续过程由脚本自动完成。

--------------------------------
六、脚本自动流程概览（V12.4）
--------------------------------
1. 图书馆页面（lib.ctgu.edu.cn）
   - 从 URL 参数中读取学号、密码、关键词、模式和抓取条数
   - 立即跳转统一身份认证登录页

2. 统一身份认证页（ids.ctgu.edu.cn）
   - 若已登录：自动识别“个人中心”，跳回图书馆
   - 若未登录：自动填入学号和密码，需手动点击“登录”

3. 知网采集

4. ScienceDirect 采集

5. 生成 Word 报告
   
注：可能在脚本执行中需要允许浏览器的弹窗

--------------------------------
七、安全与隐私
--------------------------------
1. 学号与密码仅在本地浏览器中使用，不会发送到第三方服务器。
2. 中间采集的数据保存在 Tampermonkey 的本地存储中，生成报告后可按提示清理状态。
3. 建议在个人电脑或可信环境中使用，如有需要可以在脚本中增加对 task_pass 的清理逻辑。

--------------------------------
八、常见问题 (FAQ)
--------------------------------
1. 控制台点击按钮后没有自动操作？
   - 检查 Tampermonkey 是否安装并启用
   - 检查脚本是否启用
   - 确认图书馆和统一认证站点域名与脚本 @match 一致
   - 刷新页面

2. 只想搜爱思唯尔怎么办？
   - 在控制台点击“仅爱思唯尔”按钮即可

3. 只想搜知网怎么办？
   - 同上，在控制台点击“仅知网”按钮即可

4. 想限制只抓 5 条记录怎么办？
   - 在“抓取论文条数 (1~20)”输入框填入 5，即可限制每站最多 5 条。

5. 采集时页面有时短暂停留是正常的吗？
   - 是的，脚本需要等待 CNKI 的详情块和 SD 的摘要内容加载完成
   - 期间会显示“正在收集…”的小提示，请耐心等待
